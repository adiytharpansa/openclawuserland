# Memory Template

## MEMORY.md Entry Format

```markdown
## [YYYY-MM-DD] Topik/Proyek
**Status**: ongoing | done | archived  
**Prioritas**: high | medium | low  
**Tags**: #tag1 #tag2

- Poin penting 1
- Poin penting 2
- Keputusan yang dibuat
- Link terkait: [[path/to/file.md]]
```

## Daily Note Format (memory/YYYY-MM-DD.md)

```markdown
# YYYY-MM-DD

## 📬 Inbox
- [ ] Process emails
- [ ] Check calendar

## 💬 Conversations
### [Time] Topic
- Summary
- Decisions made

## ✅ Completed
- Task 1
- Task 2

## 📝 Decisions
- Decision 1 (rationale)

## 🧠 Learnings
- Learning 1

## 🔗 References
- Link 1
```

## Memory Categories

Use these categories to organize MEMORY.md entries:

- **Preferences** - User preferences, communication style
- **Projects** - Ongoing projects and their status
- **People** - Important people and relationships
- **Systems** - Technical setups, configurations
- **Learnings** - Lessons learned, best practices
- **Decisions** - Important decisions and rationale
